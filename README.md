# Re-Imagine Vision Benchmark

This is a repository of the paper [Do Multimodal LLMs Pay Attention to What They Actually See?](https://www.techrxiv.org/doi/full/10.36227/techrxiv.177223089.92028755/v1).

It consists of:
- RelTR for scenic graph generation;  
- Image-editing models (FLUX.2 dev, HiDream and Step1XEdit);
- A script for local work with MLLMs using ollama (ask_llm.py);
- Files with results visualization (calc_metrics_additional.ipynb, calc_metrics_cats.ipynb and calc_metrics_dogs.ipynb);
- Metas (files with MLLMs answers).
